
Simple CRUD application
____
Простое CRUD приложение, позволяющее создавать, удалять и редактировать данные работников на сервере.

Создано как пример реализации веб-приложения с использованием следующих технологий: Spring, Spring MVC, SQL, Hibernate, JSP и Maven, а так же применением аспектно-ориентированного программирования.
